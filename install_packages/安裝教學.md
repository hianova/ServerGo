# ServerGo 安裝教學

本指南將引導您完成 ServerGo 的編譯、安裝與部署流程。

## 1. 系統需求

- **操作系統**: Linux (推薦 Ubuntu 22.04+), macOS (Intel/M1), Windows (WSL2)
- **開發工具**: Rust 1.75+ (建議使用 `rustup` 安裝)
- **依賴套件**: `build-essential`, `cmake` (用於編譯某些依賴項)

## 2. 從原始碼編譯

### 下載專案
```bash
git clone https://github.com/hianova/ServerGo.git
cd ServerGo
```

### 選擇模式編譯
ServerGo 支援多種功能模式，透過 Rust feature flags 切換：

- **純快取模式 (Pure Cache - 最高性能)**:
  ```bash
  cargo build --release --no-default-features --features "pure-cache"
  ```

- **分層存儲模式 (Tiered Storage - 支援持久化)**:
  ```bash
  cargo build --release --features "tiered-storage"
  ```

編譯完成後的執行檔位於 `target/release/ServerGo`。

## 3. 跨平台編譯

如果您需要為其他平台編譯（例如從 macOS 編譯 Linux 版本），建議安裝 `cross` 工具：

```bash
cargo install cross
```

### 編譯 Linux x86_64:
```bash
cross build --target x86_64-unknown-linux-gnu --release
```

### 編譯 Linux ARM64:
```bash
cross build --target aarch64-unknown-linux-gnu --release
```

## 4. 自動化部署 (Linux)

我們提供了一個自動化腳本來將 ServerGo 安裝為系統服務。

### 建立部署目錄
```bash
sudo mkdir -p /opt/servergo
sudo cp target/release/ServerGo /opt/servergo/
```

### 使用 Systemd 管理服務
建立 `/etc/systemd/system/servergo.service`:

```ini
[Unit]
Description=ServerGo High-Performance Storage Node
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/servergo
ExecStart=/opt/servergo/ServerGo --port 6379 --id 1
Restart=always

[Install]
WantedBy=multi-user.target
```

### 啟動服務
```bash
sudo systemctl daemon-reload
sudo systemctl enable servergo
sudo systemctl start servergo
```

## 5. 叢集與治理配置
 
ServerGo 支援基於 `iroh` 的 P2P 自動同步，您可以輕鬆建立分布式叢集。
 
### 啟動首個節點 (Seed Node)
```bash
./ServerGo --id 1 --port 6379
```
啟動後，日誌會顯示該節點的 `Iroh Node ID`。
 
### 加入叢集
啟動後續節點時，使用 `--peer` 參數連接至首個節點：
```bash
./ServerGo --id 2 --port 6380 --peer <NODE_1_IROH_ID>
```
 
### 治理模式調整
- **`--control-mode strict`**: 嚴格模式，僅允許初始 Leader 提案。
- **`--trust-mode full`**: 完全信任模式，在私有網路中可獲得最低延遲的廣播同步。
 
## 6. 驗證安裝

使用 `redis-cli` 進行測試：

```bash
redis-cli -p 6379 SET welcome "Hello ServerGo"
redis-cli -p 6379 GET welcome
```

如果看到 "Hello ServerGo"，恭喜您安裝成功！
